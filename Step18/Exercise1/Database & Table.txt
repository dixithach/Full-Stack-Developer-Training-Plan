1) Create a Free Supabase Account
   a) Go to https://supabase.com
   b) Click Sign Up
   c) Click Start your project
   d) Choose Sign in with GitHub
   e) log in to GitHub and authorize Supabase
   f) Once completed, you will land on the Supabase Dashboard.

2) Create a New Supabase Project
  a) In the dashboard, click New Project
  b) Give project name as family-recipes-project
  c) Create new project

3) Open the SQL Editor (Web-Based)
  a) From the left sidebar, click SQL Editor
  b) Click New query

4) Create the family_recipes Table

5) Write SQL to Create the Table
  a) CREATE TABLE family_recipes (
  id SERIAL PRIMARY KEY,
  title TEXT NOT NULL,
  instructions TEXT NOT NULL
     );
 b) Copy and paste the following SQL into the editor

6) Run the Query
 a) Click Run (or press Ctrl + Enter)
 b) You should see a success message like: “Success. No rows returned.” 


7) View the Table
 a) Verify the Table in the UI
 b) Click Table Editor from the left sidebar 
 c) Find the family_recipes
 d) Click on it
 e) You will see columns like id,title,instructions 

8) Manually Insert One Recipe Using the UI
 a) Insert Data via Table Editor
 b) While inside family_recipes, click Insert row
 c) fill the fields like title,instructions
 d) Click save

9) Confirm the Inserted Recipe
 a) Verify Data
 b) You should now see one row in the table
 c) | id | title                     | instructions                                                         |
    | 1  | Grandma’s Vegetable Curry | Chop vegetables, sauté with spices, add water, and cook until tender |
