1) Open Your Supabase Project
 a) Go to https://supabase.com
 b) Log in to your account
 c) Click on your project name from the dashboard

2) Open the SQL Editor
 a) In the left sidebar, click SQL Editor
 b) Click New query
 c) create extension if not exists vector; copy this sql command and paste it in sql editor
 d) Click run you should see Success. No rows returned message

3) Add the embedding Column to family_recipes
 a) alter table family_recipes
    add column embedding vector(384);

4) vector → data type provided by pgvector
   384 → number of dimensions in the embedding
   384 is commonly used by small AI embedding models
   alter table family_recipes → modifies the existing table
   add column embedding → adds a new column named embedding
   vector(384) → stores AI embeddings with 384 dimensions


5) Run the sql to view
 a) select column_name, data_type
    from information_schema.columns
    where table_name = 'family_recipes';
